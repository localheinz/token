Provides a simple read-only abstraction for a token and a sequence of tokens, inspired by [`friendsofphp/php-cs-fixer`](http://github.com/FriendsOfPHP/PHP-CS-Fixer).

## Installation

Run

```
$ composer require localheinz/token
```
